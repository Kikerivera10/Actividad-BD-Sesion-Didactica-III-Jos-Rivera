# Sistema de Gestión de Biblioteca (Node.js + MongoDB)

## Descripción
Proyecto que implementa una base de datos MongoDB con 5 colecciones y un módulo CRUD en Node.js usando Express y Mongoose para gestionar una biblioteca digital.

## Instrucciones para ejecutar
1. Clona este repositorio:
   `git clone https://github.com/Kikerivera10/Actividad-BD-Sesion-Didactica-III-Jos-Rivera.git`
2. Instala las dependencias:
   `npm install`
3. Configura las variables de entorno:
   Crea un archivo `.env` en la raíz del proyecto y añade:
   `MONGO_URI=tu_cadena_de_conexion_a_mongo`
4. Ejecuta el proyecto:
   `npm start`
